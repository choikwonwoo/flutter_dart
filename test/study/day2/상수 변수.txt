# 상수 변수 - const, final

### const : 컴파일 타임 상수 변수

- 톱 레벨이나  함수 내에 지역 변수로 선언할 수 있지만, 클래스에 선언할 때는 static변수로만 선언 할 수 있다.
- const 변수는 컴파일 단계에서 상수가 되므로 초기값이 반드시 있어야 하며 이후에 변경이 불가하다.

### final : 런타임 상수 변수

- 어디서나 선언이 가능하다.
- const와 마찬가지로 값을 바꿀 수 없지만 초기값 대입을 나중에 해도 된다.

### 상수변수와 문자 템플릿

- const로 지정한 string 타입 상수 변수에 문자열 템플릿으로 값을 대입할 때는 템플릿 내부에도 컴파일 타임 상수를 사용한 상수 변수만 들어가야 한다.
