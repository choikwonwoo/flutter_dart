# 널 포인트

### 널 허용과 불허

```dart
main(){
	int a1 = 10;
	int? a2 = 20;
	// 위는 널 불허 아래는 널 허가
}
```

- 다트에서 모든 타입은 객체라 초기화하지 않는 경우 null 값이 들어간다 따라서 지연 초기화를 목적으로 하려면 널 타입을 허용해야 한다
- 위의 내용은 톱 레벨이나, 클래스 변수에서 해당한다. 함수 내의 지역 변수는 해당하지 않는다.
- var 타입은 널 허용을 붙일 수 없다. null 값이 들어가면 dynamic으로 추론된다.
- dynamic 타입도 마찬가지로 널 허용을 붙일 수 없다. 의미가 없기 때문이다.
- int? → int 로 자동 형 변환 가능 역으로는 불가능
- 역으로는 as 를 사용하여 명시적으로 형 변환 가능

### late 연산자

- 널 불허 변수는 late 연산자를 이용하여 지연 초기화 시켜 준다.

### 널 안정성 연산자 !

- 널 인지 아닌지 점검 할 때는 변수 명 뒤에 !연산자를 사용한다
- 만약 해당 변수가 널 값을 가지고 있으면 에러를 발생 시킨다.

### 멤버 접근 연산자 ?. , ?[]

- 널 허용 객체나 리스트의 멤버에 접근할 때는 ?.나 ?[] 연산자를 사용해야한다.
- 이는 널이 아닐 경우만 접근하면 널이면 멤버 접근이 아닌 null 값을 반환한다.

### ??= , ??

- 널 허용 변수에  널이 아닌 값 만을 대입하고 싶다면 ??=를 사용한다.
??= 연산자는 오른쪽의 값이 null 이면 대입하지 않고 아니라면 값을 대입한다
- 널 허용 변수가 널일 때 대체할 값을 지정하고 싶다면 ??연산자를 사용한다
??연산자는 null 이 아닌 경우 오른쪽 값을 대입하고 반대의 경우 왼쪽 값을 대입한다.
